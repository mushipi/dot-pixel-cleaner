# Dot Pixel Cleaner

生成AIの「ドット絵風」画像を、真のピクセルグリッドに整列した本物のドット絵に変換するツール。

- 単一 `index.html` のみで完結（ビルド不要、外部依存なし）
- 完全クライアントサイド処理（画像は一切外部送信されない）
- グリッド自動検出（自己相関＋分散最小化）＋手動調整UI
- 最頻色/中央値によるセル代表色抽出、median cut減色、孤立1pxクリーンアップ
- 等倍／2x/4x/8x/16x PNG書き出し

## 使い方

`index.html` をブラウザで直接開くだけ（サーバ不要）。画像をドラッグ&ドロップするか、クリックしてファイルを選択する。

## 既知の制限

- 画像全体で単一のグリッドピッチ／オフセットを仮定する。生成AI画像内で擬似ピクセルの間隔が場所により揺れている場合の局所補正には対応していない（v1の既知の制限）。
